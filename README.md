This project focuses on the design and simulation of a **Cylindrical Reflector Backed Antenna** optimized for the **1 GHz** frequency range. The implementation utilizes the **MATLAB Antenna Toolbox** to analyze critical performance metrics such as directivity, impedance, and radiation patterns.

## üìä Key Results & Analysis

The repository includes several visual results from the simulation:
- **Directivity:** Analysis of the antenna's ability to focus energy in a specific direction.
- **S-Parameters:** Measurement of input return loss and bandwidth.
- **Radiation Patterns:** 3D and 2D (Azimuth) plots of the antenna's field strength.
- **Current Distributions:** Visualization of the surface currents on the reflector and dipole element.
